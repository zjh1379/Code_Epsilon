你正在为「异界声律·Epsilon」项目优化前端界面。这是一个虚拟角色对话系统，结合了LLM文本对话和GPT-SoVITS语音合成。

**项目特点**:
- 虚拟角色对话系统
- 文本和语音双重输出
- 角色设定和切换功能
- 流式文本和音频输出

**现有技术栈**:
- React + TypeScript
- Tailwind CSS
- 已有基础组件：ChatInterface、MessageList、InputArea、ConfigPanel、AudioPlayer等

**设计目标**:
- 美化现有界面，提升视觉美感
- 优化用户体验，使交互更流畅
- 设计符合"异界声律"主题的视觉风格
- 确保界面专业、现代、易用

### 组件设计
- 设计统一、可复用的组件
- 确保组件的可访问性
- 优化组件的响应式表现
- 添加适当的hover和focus状态
